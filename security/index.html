Encryption

Data at Rest

  • ✓ AES-256 encryption for all stored documents
  • ✓ Encrypted database backups
  • ✓ Encrypted vector embeddings

Data in Transit

  • ✓ TLS 1.3 for all connections
  • ✓ HTTPS enforced everywhere
  • ✓ Certificate pinning in mobile apps

Infrastructure Security

  • Hosted on SOC 2 certified cloud infrastructure
  • Network isolation with VPCs and firewalls
  • DDoS protection and WAF
  • Regular vulnerability scanning
  • 24/7 infrastructure monitoring

Access Control

  • Role-based access control (RBAC)
  • Multi-factor authentication available
  • SSO/SAML for enterprise accounts
  • Audit logs for all access
  • Automatic session timeout

AI Model Security

Your data is never used for training. We do not use customer documents or queries to train AI models. Your information is used solely to provide the service to you.

  • API calls to AI providers use encrypted channels
  • No persistent storage of queries at AI providers
  • Self-hosted option eliminates third-party AI access
  • Local LLM support for air-gapped deployments

Penetration Testing

We conduct regular penetration tests through independent security firms. Our last pentest was completed in Q4 2024 with no critical findings.

Incident Response

We maintain a comprehensive incident response plan:

  • 24/7 security monitoring
  • Defined escalation procedures
  • Customer notification within 72 hours of confirmed breach
  • Post-incident review and remediation
